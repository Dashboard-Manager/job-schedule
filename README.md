## Deployment

To deploy this project run

```bash
  docker-compose up --build -d
```

Go to site:

```bash
  0.0.0.0:8000
```

Ready :)

## Run Locally

Clone the project

```bash
  git clone https://github.com/kwiats/job-schedule job-schedule
```

Go to the project directory

```bash
  cd job-schedule
```

Install pipenv

```bash
  pip install pipenv
```

Install dependencies

```bash
  pipenv install --system
```

Go to the shell

```bash
  pipenv shell
```

Start the server

```bash
  ./manage.py runserver
```
